        <KeyboardShortcut> O </KeyboardShortcut>

        <AddedFilenameEnding> orc </AddedFilenameEnding>
        
        

        <Description>

DESCRIPTION FOR "outside cover reuse"

Image shows an instance of a reused manuscript fragment in binding, where the reuse is for the outside cover of the document. This can be a front cover image or a back cover image. It can even be an image containing the spine, but ONLY IF the outside wrapping is visible and contains text/decoration/illumination on its surface (or visible through its surface). It can also be an image of all three (front cover, back cover, and spine) where some text/decoration/illumination is visible. Once again, if you can see text/decoration/illumination through the cover - i.e. it's on the same material piece, but on the opposite side and visible through the cover surface - this 'O' should be used.

Note that this classification should only be used when the cover is on the book. If it's not on the book, the 'not in situ coVer' classification should be used. 
        </Description>

[Classification c2,] "undeR cover reuse"

        <DisplayName> undeR cover reuse </DisplayName>
        <KeyboardShortcut> R </KeyboardShortcut>
        <DirectoryName> Under_Cover_Reuse </DirectoryName>
        <AddedFilenameEnding> ucr </AddedFilenameEnding>
        
        

        <Description>

DESCRIPTION FOR "under cover reuse"

Image shows an example of reuse under the front cover or the back cover, a situation that can happen if: part of the cover has ripped away to show a document underneath; if the cover is too small to cover, e.g. a flyleaf; OR something similar.

A couple of notes:
  1) If there is reuse on the main, outside cover and such under-cover reuse, the image should be both classified as 'O' - for 'Outside cover reuse' - and as 'R' - for 'undeR cover reuse';
  2) If the only part of the outside cover that is no longer obscuring text/decoration/illumination underneath is on the spine, 'P' - for 'sPine protection reuse' - should be selected.

The current classification, ('R'), should also be used if part of front matter (especially the paste-down) is torn away showing some text/decoration/illumination on a reused piece of writing material.

Book Gutter Tears
If the book gutter has the tear, one's best judgement should be used to choose this classification ('R', classification, or C - for small Connect, strength, guard. If you're unsure which of two three you should choose with a book-gutter tear, use this classification ('R').
Though, technically, some reuse seen through the book gutter could be 'sPine protection reuse', that classification should never be used with book gutter tears, because of the definition for 'P', "[reuse visible] because part or all of the spine portion of the outer cover is missing" (emphasis added). 
        </Description>

[Classification c4,] "Front-back matter reuse"

        <DisplayName> Front-back matter reuse </DisplayName>
        <KeyboardShortcut> F </KeyboardShortcut>
        <DirectoryName> Front-back_Matter_Reuse </DirectoryName>
        <AddedFilenameEnding> fmr </AddedFilenameEnding>
        
        

        <Description>

DESCRIPTION FOR "front-back matter reuse"

Image shows text from previous use on one of a couple types of defined folios. First type: the reused document can be on a pastedown - whether it's still pasted down onto the board part of the binding, partially folded back, now unpasted but still directly after the board part of the binding, or included after a new flyleaf or flyleaves with both sides of what was previously the pastedown now available for view. The reused document can also be on a flyleaf. Usually, in an original binding, this is the first folio after the board part of the binding (and usually also the pastedown, which combines with the board to make one block). However, there can be more than one flyleaf used with reuse - corresponding to the first, second, (third, fourth, ...) leaves after the board binding/pastedown block. If the pastedown is unpasted, it can be argued that the pastedown is now a flyleaf, and the original flyleaf becomes a second flyleaf. The important points are:
  1) that the pastedown or flyleaf NOT contain ONLY material that is part of the new codex document nor of the title, description, cataloging, pen trials, etc. for the new codex document; and
  2) that the pastedown or flyleaf contain text/decoration/illumination. 
        </Description>

DESCRIPTION FOR "reuse cover wraparound"

Image is of text/decoration/illumination that is a continuation of the outside-cover text, which continuation wraps around the cover board. Note that this should be used when you are directly seeing the wraparound on the image specifically meant to show the surface where it wraps around or when you see the wraparound on the opposite surface and there is no other example of reuse - either the binding board itself or the paste-down that's still pasted down to the binding board or the page directly after either of the previous two. If you are seeing the head, tail, or (that other one where you see the edges of all the codex's pages at the same time), you should also use 'W'. (This would be a good place to have example images, if I get there.) In the case where all of one and a part of a second page are shown - i.e. the book gutter and some material past it are shown - there are two cases:
  1) if such wraparound is an edge (some edges) of the main page/surface being shown, use this classification ('W')
  2) if such wraparound is an edge (some edges) of the opposite, partial page, still use this classification ('W'). When more than one classification may be used, you should put reuse wIth other Interesting and then also across book Gutter reuse. 
        </Description>
